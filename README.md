# Mind-Booking

### This is Booking app like Goibibo


## Frontend

- React

## Backend

- Nodejs
- Expressjs
- mongoose

## Connection of front end and backend

- Proxy


## Authentication
 - jason web token for login
 - Bcryptjs for storing password in encoded form in db.
 
## Database

- Mongodb
